HyperGaussian: Referring 4D Gaussian Splatting

A unified framework for dynamic 4D Gaussian reconstruction and training-free referring understanding.

Code Project Paper

Abstract

We study Referring 4D Gaussian Splatting (R4DGS) for realistic query understanding in dynamic scenes, including temporally varying, multi-target, and distractor queries. HyperGaussian couples generalized dynamic Gaussian reconstruction, entity-centric scene structuring, and training-free referring inference. It augments dynamic Gaussians with explicit temporal support, builds a shared entitybank, decomposes complex natural-language queries into trackable phrases and constraints, and performs query-conditioned grounding without retraining scene-specific representations.
